Cubic Bézier Curve Showcase

This is a showcase of the De Casteljau algorithm for presentation purposes. It renders a cubic Bézier curve with four control points. You can show/hide the individual segmentations and adjust the resolution of the final Bézier curve. The domain of the curve is between 0 and 1 which you can sample with the t value slider. You can also adjust the curve by grabbing the red control points and move them around. The purple circle (last segment) sits on top of the Bézier curve and therefore traces ("draws") it. The implementation is done with pure JavaScript.
